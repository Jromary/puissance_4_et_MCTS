# puissance_4_et_MCTS
Le but du projet est d’écrire un programme pour jouer au puissance 4 contre l’ordinateur.

## Programmeurs
 - Angela Ipseis
 - Julien Romary

## compilation
```sh
git clone https://github.com/Jromary/puissance_4_et_MCTS.git
cd puissance_4_et_MCTS
mkdir build
cd build
cmake ..
make
```
